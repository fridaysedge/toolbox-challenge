How to Play Memory

  1. Press the "New Game" Button.
  2. Select any two tiles on the game board.
  3. If the two cards match, they stay face up.
  4. If they don't match, they get turned back over.
  5. Remember what was on each card and where it was.
  6. The game is over when all the cards have been matched.
  7. You can restart a game at any time by pressing the "New Game" button.
